## 🎨 **FUSION DESIGN SYSTEM STANDARDS**

### **Required Variants (5 Standard)**
1. **`default`** - Standard shadcn styling
2. **`premium`** - Leonard-yellow/Lahoma-orange gradients  
3. **`glass`** - Glassmorphism with backdrop-blur
4. **`tournament`** - Dark theme with gold accents
5. **`minimal`** - Clean, minimal styling

### **Required Sizes (4 Standard)**
1. **`sm`** - Small size variant
2. **`md`** - Medium (default) size
3. **`lg`** - Large size variant  
4. **`xl`** - Extra large size variant

### **Gun Club Presets (Context-Specific)**
- Tournament management components
- Range operation controls
- Member portal interfaces
- Schedule/event management
- Safety and rules displays

### **Required Technologies**
- **CVA** (Class Variance Authority) for variants
- **Framer Motion** for animations
- **Tailwind CSS** for styling
- **Radix UI** for base primitives
- **TypeScript** for type safety

# Project Guidelines

This documentation provides comprehensive guidance for working with the Boise Gun Club v4 project.

## Tech Stack

- **Next.js 15.3.2** with App Router
- **React 19** with modern patterns  
- **TypeScript** with strict mode
- **Tailwind CSS 4.0** with CSS-first configuration
- **Framer Motion** for animations
- **Storybook 9.0.15** for component documentation
- **Jest** with React Testing Library for testing

## Development Commands

### Core Development
```bash
npm run dev          # Start development server
npm run build        # Build production version  
npm run start        # Start production server
npm run lint         # Run ESLint
npm run type-check   # Run TypeScript checking
```

### Testing & Analysis
```bash
npm test             # Run Jest tests
npm run test:watch   # Run tests in watch mode
npm run storybook    # Start Storybook dev server
npm run analyze      # Bundle size analysis
```

## Component Enhancement Guidelines

When creating or enhancing components:

1. **Use the CVA Pattern** - Implement variants using Class Variance Authority
2. **Include Fusion Variants** - Add premium, elite, and glass variants where appropriate  
3. **Follow Animation Standards** - Use consistent timing and easing
4. **Implement Accessibility** - Support keyboard navigation and screen readers
5. **Document in Storybook** - Create comprehensive stories with all variants

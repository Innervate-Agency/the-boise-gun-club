## üö® **Common Mistakes to Avoid**

1. **White text on white background**
   - ‚ùå `text-white` in light mode
   - ‚úÖ `text-gray-900 dark:text-white`

2. **Invisible badges on gradients**
   - ‚ùå `bg-green-500 text-green-500` on green gradient
   - ‚úÖ `bg-white/20 text-white border-white/30` on any gradient

3. **Square card corners**
   - ‚ùå Default card styling
   - ‚úÖ `rounded-xl overflow-hidden` + `rounded-t-xl` for header

4. **Inline gradient styling**
   - ‚ùå `style={{ background: 'linear-gradient(...)' }}`
   - ‚úÖ Tailwind gradient classes or CSS variables

5. **Wrong terminology**
   - ‚ùå "Rifle range" or "paper targets"
   - ‚úÖ "Shotgun sports" and "clay targets"

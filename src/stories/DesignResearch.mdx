# Stripe + ClickUp Design Research

## Research Findings

### Stripe's Premium Micro-Interactions

Based on analysis of stripe.com, here are the key patterns that make their components feel premium:

#### Animation Timing
- **Duration**: `150ms` for micro-interactions
- **Easing**: `cubic-bezier(0.215, 0.61, 0.355, 1)` - Stripe's signature easing function
- **Hover delays**: Instant response with smooth transitions

#### Button Styling
- **Border radius**: Consistent `8px` 
- **Shadows**: `0 2px 4px -1px rgba(50,50,93,.25)` for depth
- **Scale effects**: Subtle `scale(1.02)` on hover
- **Focus states**: Clean ring indicators

#### Visual Hierarchy
- **Typography**: Medium font weights, never too heavy
- **Spacing**: Generous white space and consistent padding
- **Subtle gradients**: Professional color transitions
- **Refined shadows**: Layered for realistic depth

### ClickUp's Gradient Magic

While CSS details were limited, ClickUp's visual patterns include:

#### Glassmorphism
- **Backdrop blur**: Creates depth and modern feel
- **Transparency layers**: Multiple opacity levels
- **Color overlays**: Vibrant but tasteful

#### Animation Patterns
- **Smooth tab transitions**: Seamless state changes
- **Interactive elements**: Responsive hover states
- **Dynamic content**: Smooth filtering and sorting

## Technical Implementation

### CSS Custom Properties (Enhanced)
```css
/* Stripe-inspired timing */
--transition-fast: 150ms cubic-bezier(0.215, 0.61, 0.355, 1);
--transition-smooth: 300ms cubic-bezier(0.215, 0.61, 0.355, 1);

/* Professional shadows */
--shadow-stripe-sm: 0 2px 4px -1px rgba(50,50,93,.25);
--shadow-stripe-md: 0 4px 8px -2px rgba(50,50,93,.25);
--shadow-stripe-lg: 0 8px 16px -4px rgba(50,50,93,.25);

/* Glassmorphism layers */
--glass-subtle: rgba(255, 255, 255, 0.05);
--glass-medium: rgba(255, 255, 255, 0.1);
--glass-strong: rgba(255, 255, 255, 0.15);
```
